# TPRealidadAumentada

Aplicación de Realidad Aumentada

Crear un prototipo de una aplicación de realidad aumentada con alguna de las siguientes características:
Detección de imágenes
Detección de planos
Detección de Rostros
Spawneo y manipulación de objetos 3D
UI interactiva ( menús, controles de objetos )

Dicho Prototipo actual cuenta con las siguientes caracteristicas de las mencionadas:
- Detección de planos
- Spawneo y manipulación de objetos 3D
- UI interactiva ( menús, controles de objetos )

Controles:
Al Abrir la App tendrás un boton para desplegar un menú del cual podrás seleccionar el objeto a Instanciar, para instanciarlo deberás apuntar con la camara a un PLano y que este se considere un plano a su vez (asegurarse de mantener una buena iluminacion y la camara limpia), despues tendrás que pulsar en el area detectada para poder spawnear el objeto, este mismo una vez spawneado podrás manipularlo de manera interactiva, ajustando su posicion, rotacion y tamaño.
